### Question 1

The Design Recipe in this course consists of six steps numbered 0 through 5. Below is a scrambled list of step descriptions:

1. Examples and Tests
2. Method Signature and Purpose Statement
3. Implementation, Testing, and Refinement
4. Understand and Restate the Problem
5. Skeleton and Method Template
6. Data Definitions

Which option lists the six descriptions in the **correct order**, from Step 0 to Step 5?

* **A)** 4, 6, 2, 1, 5, 3
* **B)** 4, 2, 6, 1, 5, 3
* **C)** 4, 6, 1, 2, 3, 5
* **D)** 6, 4, 2, 1, 5, 3
* **E)** 4, 1, 6, 2, 3, 5

**Your answer:** _A_
