### Question 1

The `Student` class declares which set of fields?

* **A)** `String name`, `int studentID`, `String[] courses`
* **B)** `String name`, `String studentID`, `ArrayList<String> courses`
* **C)** `String name`, `String studentID`, `Course[] courses`
* **D)** `Student[] friends`, `String studentID`, `Course course`
* **E)** Just a single `String name` field

**Your answer:** _B_

### Question 2

The Javadoc on `Student.addCourse(String course)` reads:

```
Returns the course name if successfully added, otherwise "-99".
```

What does this method return when the student is already enrolled in `course`?

* **A)** `null`
* **B)** `false`
* **C)** The empty string `""`
* **D)** The literal **`String`** `"-99"`
* **E)** The integer `-99`

**Your answer:** _D_

### Question 3

Which body correctly implements `Student.addCourse(String course)` so it adds the course only if it is not already on the list, returning the course name on success and `"-99"` otherwise?

* **A)**
    ```java
    public String addCourse(String course) {
        courses.add(course);
        return course;
    }
    ```
* **B)**
    ```java
    public String addCourse(String course) {
        if (courses.contains(course)) {
            return "-99";
        }
        courses.add(course);
        return course;
    }
    ```
* **C)**
    ```java
    public String addCourse(String course) {
        if (!courses.contains(course)) {
            return "-99";
        }
        courses.add(course);
        return course;
    }
    ```
* **D)**
    ```java
    public String addCourse(String course) {
        courses[courses.size()] = course;
        return course;
    }
    ```
* **E)**
    ```java
    public String addCourse(String course) {
        courses.add(0, course);
        return "-99";
    }
    ```

**Your answer:** _B_

### Question 4

Which statement best captures the relationship between `Student` and `StudentManagement`?

* **A)** `StudentManagement` extends `Student` (inheritance).

### Question 9
